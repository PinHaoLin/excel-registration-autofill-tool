# Design

## Document Format
The deliverable is a `.docx` file in `docs/`, created as a polished operator reference guide for non-technical users.

## Style Preset
Use the Documents skill `compact_reference_guide` preset:

- US Letter portrait.
- 1 inch margins.
- Calibri 11 pt body text.
- Blue heading hierarchy.
- Fixed-width tables with readable cell padding.
- Real numbered and bulleted lists.

## Content Requirements
The Word guide must cover:

- Tool purpose.
- Before-use checklist.
- Main screen overview.
- Step-by-step workflow.
- Sample input text.
- Excel output mapping.
- Windows unknown-source / trust prompt guidance.
- Troubleshooting.
- Safe usage recommendations.

## Visual Aids
Use simple Word-native visuals rather than Mermaid source blocks:

- Numbered workflow table.
- Interface area table.
- Checklists.
- Column mapping tables.

## Validation
Render the generated `.docx` to page images and visually inspect the pages when rendering support is available.
