# Design

## Version Source
The release artifact name is generated by `electron-builder` from `package.json`:

`Excel自動填寫工具-${version}.${ext}`

Updating `package.json` and `package-lock.json` to `0.1.1` makes the packaged executable output as `Excel自動填寫工具-0.1.1.exe`.

## Documentation Sync
Markdown and Word user guides SHALL reference the current packaged executable name so general users can identify the correct version.

The Word guide SHALL continue to be generated by `scripts/build_word_user_guide.py` to keep the `.docx` repeatable.

## Release Output
The final portable executable SHALL be produced under `release/`.

The release folder remains ignored by Git, so GitHub stores source code and documents while local distribution uses the rebuilt executable artifact.
